# Luke's Portfolio

Static multi-page portfolio built with vanilla HTML, CSS, and Javascript.

## Features
- Responsive Design (mobile-first TBA)
- Dark Mode Toggle (vanilla JS)
- Served locally with HTTPS using mkcert(TBA)

## Local Development
1. Open in VS Code
2. Use Live Server extension (or your preferred method)
3. FOr HTTPS: Run 'mkcert localhost' -> point Live Server or server config to the generated cert/key

**NOTE** local certs are machine-specific and **never** commited tot he repo.
