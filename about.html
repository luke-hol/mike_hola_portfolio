Project Nebula

Making awesome things a little less painful since 2025

Vibe Coding Till I Can Actually Code

Project Nebula is an unborn bastardized "fundamentals" project that is also AI assisted. I built it because I got tired of knowing enough syntax to debug and add features to code, but never saw how they fit together. Almost like I needed a "blue-print" to figure out smaller puzzle pieces together, and then stich them into something big that works.

Created mostly with a post-high haze as a distraction while fighting to be sober at 2 a.m.
